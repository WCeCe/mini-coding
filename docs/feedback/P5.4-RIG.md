## 方案摘要

### 模块布局

```
mini_coding_agent/rig/
  store.py    # SQLite schema（files / symbols / imports）+ RigStore
  build.py    # ast 全量扫描 *.py，忽略 IGNORED_PATH_NAMES
  query.py    # RigQuery：by_symbol / by_file / one_hop_neighbors
  __init__.py
```

**存储路径**：`<repo>/.mini-coding-agent/rig.db`

### 图谱实体（MVP）

| 表 | 内容 |
|----|------|
| `files` | 相对路径、mtime |
| `symbols` | 函数/类/方法：name、kind、lineno、qualname |
| `imports` | file_path → module、imported_name |

### CLI

```bash
python -m mini_coding_agent.cli rig build [--cwd .]
```

`main()` 在解析 Agent 参数前检测 `argv[0] == "rig"`，委派 `rig_main()`。

### locate 集成

```
run_locate
  → RigQuery.for_repo(agent.root)  # 无 rig.db 则 None
  → 符号：先 by_symbol；无命中再 search（rg 回退）
  → 路径：by_file + one_hop_neighbors；再 read_file
  → data.used_rig 标记是否使用图谱
```
